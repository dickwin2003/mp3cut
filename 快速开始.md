感谢 GitHub Copilot 免费计划对本项目开发的支持！

## 一步启动

1. 安装 [.NET 6.0 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)
2. 下载本项目源码
3. 命令行进入项目目录，执行：
   ```bash
   dotnet restore
   dotnet run
   ```

或用 Visual Studio 2022 打开 `SimpleMp3Tool.csproj`，按F5运行。

## 主要功能
- 拖拽/选择MP3文件
- MP3播放、暂停、停止，显示进度
- 按时间剪切、按段数剪切
- 多文件合并
- 输出路径自定义
